# Vibe_Show
Este projeto propõe o desenvolvimento de um software moderno e interativo voltado para a venda de ingressos de um show musical, oferecendo aos usuários uma experiência completa desde a escolha do evento até a definição exata do local onde irá ficar durante a apresentação.

A plataforma permitirá que os usuários explorem informações planejadas sobre o show, como dados, horário, local, artistas participantes, repertório esperado e outras atrações relacionadas. Um dos principais diferenciais do site será a visualização do espaço do evento, com um mapa interativo que mostra setores como pista, camarotes, área VIP e arquibancadas.

Durante o processo de compra, o usuário poderá selecionar não apenas o tipo de ingresso, mas também visualizar exatamente onde ficará posicionado no evento. O sistema exibirá a oferta do local em tempo real, diminuindo as áreas disponíveis, ocupadas e os preços de cada setor, facilitando a escolha de acordo com preferência e orçamento.
