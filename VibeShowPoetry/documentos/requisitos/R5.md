# Cancelamento e reembolso

**COMO** Cliente,  
**QUERO** cancelar minha compra  
**PARA** obter reembolso quando necessário.  

**Critérios de Aceitação:**

- O sistema deve permitir solicitar cancelamento  
- O sistema deve validar regras de prazo para reembolso  
- O sistema deve processar o reembolso automaticamente  
- O usuário deve receber confirmação do cancelamento
