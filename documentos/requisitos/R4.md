# Cadastro de usuário

**COMO** Usuário,  
**QUERO** criar uma conta no sistema  
**PARA** facilitar minhas compras e acessar meus ingressos.  

**Critérios de Aceitação:**

- Deve ser possível realizar cadastro com e-mail e senha  
- O sistema deve validar os dados informados  
- O usuário deve conseguir fazer login após o cadastro  
- Deve ser possível recuperar a senha
