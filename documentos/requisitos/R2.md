# Escolha de assentos/setores

**COMO** Cliente,  
**QUERO** visualizar e escolher o local onde ficarei no show  
**PARA** garantir uma melhor experiência durante o evento.

**Critérios de Aceitação**

- Deve ser possível visualizar um mapa do local do evento  
- Deve ser possível selecionar setores (pista, VIP, camarote, etc.)  
- O sistema deve indicar quais lugares estão disponíveis ou ocupados  
- O sistema deve mostrar o preço de cada setor  
- Cada setor deve ser diferenciado por cores
